Project Development Log

This page summarizes major user directions and key assistant suggestions rather than reproducing every draft.

  • User requested a neutral website ranking the top 100 Trump accomplishments or impacts, good and bad, by impact.
  • User requested supportive and critical source links instead of ideological labels.
  • User requested transparent weighting and a ranking-details widget.
  • User identified that a people/economic-heavy weighting underweighted USAID, WHO withdrawal, PEPFAR, courts, and institutional actions.
  • Assistant suggested the current six-factor weighting: People 22%, Economic 18%, Permanence 22%, Duration 13%, International 13%, Historical Significance 12%.
  • User requested governing documents to prevent regressions across versions.
  • User approved a JSON-driven workflow so future builds preserve the master data source.
  • This package expands all 100 entries and should be reviewed against the governing documents before public launch.

Production Candidate v1.0. Rankings measure estimated impact, not approval or disapproval.
